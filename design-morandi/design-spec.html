Design System · 设计规范

宝盒设计规范

一份把这一轮所有设计决定收拢到一处的交付文档——配色、字体、图标、标志性元素、卡片、心情系统、命名、页面与语气。给设计与开发一份可对照落地的单一依据。

产品 宝盒 / Nesio · 本地优先的个人记忆 助理 念念 / Nessa 基调 莫兰迪低饱和 · 冷静而温暖 版本 v1
01

设计原则

六条贯穿全局的判断标准。任何新界面拿不准时,回到这里对齐。

回溯,不预测
把你存过的、感受过的,在对的时候轻轻翻给你;不替你决定未来。这是产品的立身之本。
冷静而温暖
像个理性可靠、又记得你一切的老友。不煽情、不催促、不营销腔。
少即是多
留白、克制、层级清楚。宁可少一个元素,不多一分噪音。
无 emoji
统一的线性图标语言承载所有符号,不用彩色 emoji,保持高级与一致。
分类为主,来源为辅
记忆先按含义分类呈现,来源(手记/邮件/Notion…)退为辅助徽章。
站在盒子这边
文案视角始终是"念念替你收着、替你记着",而非系统功能自述。
02

品牌与念念

产品叫 宝盒 / Nesio;助理叫 念念 / Nessa。对话入口直接以她命名(原「问一问」→「念念」);首页问候、回溯与洞察里所有"它说的话",也都是念念的口吻。

念
念念 / Nessa

念念——"念念不忘",把你的事都放在心上的那一个。
Nessa——Nesio × 记忆女神 Mnemosyne 的混血,字面即"记忆的那一个",与品牌同声同源。

她的脸

头像那颗哑光小球就是念念的脸——白核 + 淡彩光晕(见「标志性元素」)。同一颗球也用于记忆罐的收藏。

口吻示例

被叫到
"在的。想找什么,我替你翻。"
回溯
"一年前的今天,你去看了海——还记得吗?"
收下
"收好了,不急,我替你记着。"
问心情
"此刻,你感觉——?"

两种角色:衬线承载温度与"念念的声音",无衬线承载 UI 与数据。二者分工清晰,不混用。

06

标志性元素

三个专属视觉,让宝盒一眼可认。

水晶球 · 记忆罐

质感:哑光白核 + 淡彩光晕,轻盈通透。白色径向渐变作核,边缘染一层皮肤辅助色,左上一点高光。
用途:记忆罐的收藏(核心记忆/收藏/项目),以及念念的头像。不用于普通记忆卡片——保持珍藏感。

心情波纹符号

圆角方块 + 波纹,专用于心情。刻意区别于日程的圆点——形状不同,一眼看出"这不是日程,是问你自己"。方块底色与波纹随当天情绪色变化。

玻璃透视 · App 图标

App 图标沿用现有的玻璃立方体:等距、顶面透亮、内 3 条白线撑出立体、柔光落地。保持不变。

情绪盘
Russell 四象限布局,滑选。中心大圈填当前情绪色(滑到哪变哪色,兼作实时预览),环收紧不留大片空白。长按中心可直接写下来。
落位 · 首页第一拍
心情是一天时间线的「现在」第一拍,不占额外空间。用心情波纹方块作节点,标题文字取盘里对应色——一列排下就是你的情绪光谱。

往下还有:能量条(蔫→满电)、一句话/展开写(journal-lite)、留住的时刻(历史时间线)。情绪与能量喂给「洞察 → 一周能量 / 情绪回暖」。

10

页面布局

各页的结构要点。详见对应设计稿。

今天TODAY
问候(念念口吻) → 回顾卡(去年今日/一段旧记忆) → 接下来时间线。每个节点带具体时间与倒计时;心情作「现在」第一拍。处处可点开。
记忆MEMORY
搜索 → 记忆罐(3 水晶球:核心记忆/收藏/项目) → 收纳实用入口条(40 件·估值) → 全部记忆(统一卡片瀑布流)。
洞察INSIGHTS
多 Tab。含"存入记忆"、可选角度/月/周/任意时段/分类的 filter;历史信可折叠存档回看,每屏一封。
多面镜MIRROR
照见另一面的自己——认知/人格切片,以卡片与信件体呈现。
足迹PLACES
四分类。地点=饼图 + 周/月/年 filter;世界=先国家后城市,用可含图文的 card。
念念NESSA
跟念念对话(原「问一问」)。历史记录、新对话按钮、深问模式开关;长按消息 → 存入记忆/复制/以此为题继续问。
心情MOOD
四层:情绪盘(12 色) → 能量条 → 一句话/展开写 → 留住的时刻(历史)。
设置SETTINGS
档案(温暖) vs 账户(冷静)分开;外观 = 配色(4) + 明暗(3)。去掉旧的"了解你/语气/提醒程度"等项。
记一笔CAPTURE
中心盒子 → 均匀弧展开 3 个:拍一下/说一句/收进来。记心情已移至今天页,不在此。
11

语气

念念说话的方式:冷静、温暖、克制。以下是标尺。

要
  • 像老友:"我先替你收着,不急。"
  • 回溯用回忆口吻:"还记得吗?"
  • 给 ADHD 减压:"只做下一步就好。"
  • 短句、留白、具体
不要
  • 营销腔、感叹号轰炸
  • 预测/替你决定未来
  • 系统功能自述:"已同步 3 条数据"
  • emoji、彩色、催促

一句话:产品里所有"它说的话",都是念念在跟你说。
